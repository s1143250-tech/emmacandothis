# 767 目錄說明

## 檔案分析

### uno-game.html
這是一個單人對戰的 UNO 紙牌遊戲網頁應用程式。

#### 功能列表：
1. **遊戲初始化** - 發給玩家和電腦各 7 張牌，隨機抽取第一張出牌
2. **卡牌系統** - 完整 UNO 卡牌：
   - 數字卡 0-9（顏色：紅、藍、綠、黃）
   - 特殊卡：skip（跳過）、reverse（反轉）、draw2（抽2張）
   - 萬用卡：wild（換色）、wildDraw4（抽4張）
3. **玩家互動** - 點擊卡牌出牌，點擊牌堆抽牌
4. **電腦 AI** - 隨機出牌或抽牌
5. **顏色選擇** - 出 wild 牌時需選擇顏色
6. **喊 UNO** - 剩一張牌時可喊 UNO（未喊罰2張）
7. **遊戲結束** - 判斷輸贏並顯示結果
8. **重新開始** - 可開啟新遊戲

#### 技術規格：
- 單一 HTML 檔案（含 CSS 和 JavaScript）
- 無需後端資料庫
- 支援現代瀏覽器
